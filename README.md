# Rincón Natural - Sistema de Gestión para Dietética

Aplicación completa de gestión para dietética con control de inventario, empleados, proveedores y caja.

## 🎨 Características

- **Autenticación**: Sistema de login seguro
- **Gestión de Productos**: Carga de productos con cantidades en kg, gramos o unidades
- **Empleados**: Registro con horarios de trabajo y roles (personal/empleado)
- **Proveedores**: Directorio de proveedores con contacto
- **Control de Stock**: Monitoreo de inventario con alertas de stock mínimo
- **Caja**: Registro de ventas por rango de fechas con exportación a Excel
- **Reportes**: Detalles de ventas diarias y exportación de datos

## 🚀 Stack Tecnológico

### Backend
- **Node.js** + **Express** - Servidor web
- **Mongoose** - Modelado de datos
- **MongoDB** - Base de datos
- **JWT** - Autenticación

### Frontend
- **React** + **TypeScript** - UI interactivo
- **Tailwind CSS** - Estilos
- **Vite** - Build tool
- **Axios** - Cliente HTTP
- **xlsx** - Exportación a Excel
